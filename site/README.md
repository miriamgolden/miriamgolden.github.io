# miriamgolden.com

Personal academic website for Miriam A. Golden, hosted on GitHub Pages.

## Editing the site

Each page is a standalone HTML file. To update content, edit the relevant `.html` file directly:

- `index.html` — Home page (bio, contact info)
- `books.html` — Books & edited volumes
- `articles.html` — Journal articles & book chapters
- `opeds.html` — Op-eds, blogs, commentary
- `research.html` — Working papers
- `resources.html` — Research guides for students
- `nsf-ddrig.html` — NSF DDRIG guide
- `teaching.html` — Teaching overview
- `courses.html` — Course listings
- `students.html` — Students & post-docs

## Adding files

- **PDFs** (CV, syllabi, papers): Place in `assets/files/`
- **Images** (profile photo): Place in `assets/images/`
- **CSS styling**: Edit `assets/css/style.css`

## Custom domain

The `CNAME` file tells GitHub Pages to serve this site at `miriamgolden.com`.

## Notes

- No build tools required — just plain HTML/CSS
- Changes go live within minutes of pushing to GitHub
- Free hosting via GitHub Pages with HTTPS
